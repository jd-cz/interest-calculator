# Compound Interest Calculator (Vite Web App) — Instructions

## Goal
Build a simple web app using **Vite** that calculates **compound interest** and shows the results clearly.

The app should:
- Let the user enter values (principal, interest rate, time, compounding frequency, optional monthly contribution)
- Compute final amount and interest earned
- Update results on button click
- Look clean and easy to use

## Tech Stack
- Vite
  - React (recommended if you want components)
  - TypeScript
  - Pick well-known libraries for charts and/or UI components if needed

## Inputs (Minimum)
1. Principal
2. Annual interest rate (%)
3. Years
4. Compounding frequency
   e.g. yearly (1), quarterly (4), monthly (12), daily (365)

Optional:
- Monthly contribution

## Outputs (Minimum)
Display:
- A table of year-by-year growth
- Final amount
- Total interest earned
- Total contributions (if monthly contribution is included) 
- A chart of balance over time

## UI Requirements
- Simple form layout
- Validate inputs
- Results should be formatted as currency
- Button: “Calculate”
